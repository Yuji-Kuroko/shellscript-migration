# ShellScript migration system

## About

This is ShellScript migration system for **server environment**.

## How to Use

Create migration.
Please you edit migration file in `migrations/`
```bash
./mg.sh create foobar
# 20160102195007_foobar.sh
```

Migrate.
```bash
./mg.sh migrate
# 20160102195007_foobar.sh
```
